# System Design LLD Learning Repo

This repository is organized as a learning path for Low Level Design (LLD). Start from the basics and move forward in the order below.

## Learning Path

1. [Intro to CPP](Intro%20to%20CPP/) - Start here for C++ basics, syntax, and simple problem solving.
2. [OOP's](OOP's/) - Learn encapsulation, inheritance, polymorphism, and abstraction.
3. [SOLID principles](SOLID%20principles/) - Understand how to design cleaner and more maintainable classes.
4. [Design Patterns](Design%20Patterns/) - Study Builder, Singleton, and Creational design pattern examples.
5. [functors](functors/) - Practice comparator logic and function-object based sorting examples.
6. [Intor to JAVA](Intor%20to%20JAVA/) - Build Java fundamentals and beginner project structure.
7. [IdeaProjects](IdeaProjects/) - Explore larger LLD case studies and practice projects.

## Recommended Study Order Inside `IdeaProjects`

1. [Car Parking Lot System](IdeaProjects/Car%20Parking%20Lot%20System/) - Model parking spots, tickets, vehicles, and payments.
2. [Meeting System](IdeaProjects/Meeting%20System/) - Practice scheduling, intervals, rooms, and notifications.
3. [Car Rental System](IdeaProjects/Car%20Rental%20System/) - Study branches, booking flow, payments, and decorators.

## Notes

- Generated folders such as `output/`, `bin/`, `build/`, and `target/` are ignored from Git.
- Keep source files, diagrams, and notes under the topic folder they belong to.
- If you add a new LLD topic, place it in the sequence that best matches the learning path above.

## Suggested Flow

```text
Basics -> OOP -> SOLID -> Design Patterns -> Java Basics -> LLD Projects
```

That flow will help you move from language fundamentals to real object-oriented design problems without jumping around.
